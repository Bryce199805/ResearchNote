# Markdown 公式符号



### 大括号

$$
x =
\begin{cases}
a, {\kern 12pt}if \ x>0 \\
b, {\kern 12pt}if\ x<0 \\
0, {\kern 12pt}if \ x=0
\end{cases}
$$

### 下参数

$$
\underset{n \rightarrow \infty}{lim} x_n=x
$$

### 上标记

| $\overline{a}$  | \overline{a}  |
| $\widehat{a}$   | \widehat{a}   |
| $\widetilde{a}$ | \widetilde{a} |
| $\dot{a}$       | \dot{a}       |
| $\ddot{a}$      | \ddot{a}      |
